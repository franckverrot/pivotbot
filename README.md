# pivotbot

## Setup

`stack build`


## Usage

```
λ pivotbot – A Pivotal Tracker command-line tool

Usage: pivotbot-exe (--show-story ARG | --list-stories ARG)
  Instant access to your stories

Available options:
  -h,--help                Show this help text
```

### Options

`--action` : Actions availables:

  * `list-stories`
